Welcome to Que!

New Admin User

We're going to create a superadmin user - you shouldn't need to log in as the superadmin unless you have problems down the road. Type in a really secure password, because the superadmin has permission to do anything.

Your Account

Also, you should create a user account for yourself. This will let you login and access Que. This user will be given (by default) these permissions: thing.*, block.*, token.*, and notify.*.

Well, that's it.

On the next screen, login to Que using the username and password you created. Once you've logged in, take a look at the settings page and fill in the auth tokens for the various services. Until you do this, these devices / services won't work.
